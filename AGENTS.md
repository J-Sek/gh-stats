# Project Rules

## General
- Follow the existing code style and patterns.
- Use pnpm for running project commands.
- Keep code in TypeScript unless migration is required.

## Vuetify
- **ALWAYS use MD3 typography classes** (`text-display-large`, `text-headline-medium`, `text-body-large`, `text-label-small`, etc.) instead of legacy classes like `text-caption`, `text-subtitle-1`, `text-h6`, etc. This is a hard rule — no exceptions.

## Stack
- Framework: Vue 3 + Vite
- UI Library: Vuetify
- Enabled Features: ESLint, Pinia, Vue I18n, Vuetify MCP, Vue Router, UnoCSS + Vuetify Preset
